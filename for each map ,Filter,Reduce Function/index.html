JavaScript Methods Visualizer

Array Methods:

push(): Adds one or more elements to the end of an array.

pop(): Removes the last element from an array.

shift(): Removes the first element from an array.

unshift(): Adds one or more elements to the beginning of an array.

concat(): Merges two or more arrays.

join(): Converts an array to a string.

slice(): Extracts a portion of an array.

splice(): Adds/removes elements at a specified index.

indexOf(): Finds the first index of an element.

lastIndexOf(): Finds the last index of an element.

includes(): Checks if an element exists in an array.

find(): Finds the first element matching a condition.

findIndex(): Finds the index of the first element matching a condition.

filter(): Returns all elements that match a condition.

map(): Transforms each element in an array.

reduce(): Reduces the array to a single value.

reduceRight(): Reduces the array from right to left.

sort(): Sorts the elements of an array.

reverse(): Reverses the order of elements.

forEach(): Executes a function for each element.

every(): Checks if all elements satisfy a condition.

some(): Checks if at least one element satisfies a condition.

fill(): Fills an array with a specified value.

copyWithin(): Copies a portion of an array to another location.

flat(): Flattens nested arrays into a single array.

flatMap(): Maps and flattens the result in one step.

String Methods:

charAt(): Returns the character at a specific index.

charCodeAt(): Returns the Unicode of the character at a specific index.

concat(): Joins two or more strings.

includes(): Checks if a substring exists within a string.

indexOf(): Finds the position of the first occurrence of a substring.

lastIndexOf(): Finds the position of the last occurrence of a substring.

match(): Finds matches using a regular expression.

replace(): Replaces a substring or match with a new string.

replaceAll(): Replaces all occurrences of a substring.

search(): Searches for a match using a regular expression.

slice(): Extracts a portion of a string.

split(): Splits a string into an array of substrings.

substring(): Extracts a substring between two indexes.

substr(): Extracts a substring (deprecated).

toLowerCase(): Converts a string to lowercase.

toUpperCase(): Converts a string to uppercase.

trim(): Removes whitespace from both ends of a string.

trimStart(): Removes leading whitespace from a string.

trimEnd(): Removes trailing whitespace from a string.

padStart(): Pads the start of a string with a specified character.

padEnd(): Pads the end of a string with a specified character.

repeat(): Repeats a string a specified number of times.

localeCompare(): Compares two strings lexicographically.

Function Methods:

call(): Invokes a function with a specified context.

apply(): Invokes a function with arguments as an array.

bind(): Creates a new function with a specific context.

toString(): Converts a function to a string representation.

length: Returns the number of parameters the function accepts.

name: Returns the name of the function.

arrow function: Defines a function in a shorter syntax.
